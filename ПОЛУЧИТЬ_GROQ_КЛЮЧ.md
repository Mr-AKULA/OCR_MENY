## ❌ ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### Проблема 1: "Ключ не работает"

```
Решение:
1. Проверьте что скопирован ВСЕ
   Должно быть ~100 символов, начинается с gsk_
2. Нет пробелов в начале/конце
3. Попробуйте создать новый ключ:
   - API Keys → Delete (старый)
   - Create API Key (новый)
```

### Проблема 2: "Забыл скопировать ключ"

```
Решение:
1. Откройте https://console.groq.com/
2. Нажмите API Keys
3. Нажмите на существующий ключ
4. Скопируйте его
5. Если не видите опцию копирования - удалите и создайте новый
```

### Проблема 3: "Не помню пароль от Groq"

```
Решение:
1. Откройте https://console.groq.com/
2. Нажмите "Forgot Password?"
3. Введите email
4. Придет письмо с ссылкой на сброс
5. Установите новый пароль
```

### Проблема 4: "Нет доступа после регистрации"

```
Решение:
1. Проверьте что подтвердили email
   (письмо должно быть в inbox или спаме)
2. Если письма нет:
   - Откройте https://console.groq.com/
   - Нажмите "Resend Verification Email"
   - Проверьте почту заново
```
