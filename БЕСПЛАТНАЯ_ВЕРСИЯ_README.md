### Шаг 1: Получить Groq API ключ (2 минуты - БЕСПЛАТНО!)

```
1. Откройте: https://console.groq.com/
2. Нажмите "Sign Up" (регистрация)
3. Выберите "Continue with Google" или "Email"
4. Подтвердите email
5. Левое меню → "API Keys"
6. Нажмите "Create API Key"
7. Скопируйте значение вида: gsk_xxxxxxxxxxxx...
8. Вставьте в БЛОК 3 ноутбука
```

### Шаг 2: Скопировать ноутбук в Google Colab

```
1. Откройте: https://colab.research.google.com/
2. Нажмите: File → Upload notebook
3. Выберите: menu_parser_free.ipynb
4. ИЛИ просто скопируйте содержимое в новый notebook
```

### Шаг 3: Вставить Groq API ключ

## 🆓 ПОЧЕМУ ВСЁ БЕСПЛАТНО?

### Tesseract OCR
- **Open-source** проект
- **Бесплатен** всегда
- **Установлен** в Google Colab по умолчанию
- Поддерживает **русский** язык ✅

### Groq API
- **8000 токенов в минуту** бесплатно
- Хватает для ~50 меню в минуту
- Быстрый инференс (2-3 сек на меню)
- Не требует карточки

### Google Drive
- **15 ГБ** бесплатно
- Достаточно для **тысяч меню**

### Google Colab
- **50 ГБ** свободного места
- **GPU/TPU** время ограничено, но для этой задачи не нужно
- Совершенно бесплатен

```
Решение:
1. Откройте https://console.groq.com/
2. Проверьте что ключ скопирован ПОЛНОСТЬЮ
3. Ключ должен выглядеть как: gsk_xxxxxxxxxxx...
4. Вставьте без пробелов
```

### Ошибка 2: "Google Drive not mounted"

```python

```

### Ошибка 3: "No such file or directory: photos"

```
Решение:
1. Убедитесь что создали папку: Menu_Parser/photos
2. Загрузите фото на Google Drive в эту папку
3. Или раскомментируйте БЛОК 10 для загрузки с URL
```

### Ошибка 4: "Low OCR quality - много опечаток"

```
Решение:
1. Используйте фото хорошего качества (1080+ px)
2. Хороший контраст (черный текст на белом)
3. Минимум бликов и теней
4. Текст должен быть четким
```
